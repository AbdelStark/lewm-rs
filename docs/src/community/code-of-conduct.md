# Code of conduct

[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Full text in
[`CODE_OF_CONDUCT.md`](https://github.com/AbdelStark/lewm-rs/blob/main/CODE_OF_CONDUCT.md).
